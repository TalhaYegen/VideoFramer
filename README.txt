There is matlab code in .m format and the app itself. 
 - The app requires the matlab runtime which is automatically setup by the exe file.

What does it do?

It slices the video frame by frame to a folder selected by the user in preselected format such as jpg, jpeg, png, tiff.

How is it work?

1. Open the program. 
2. Select the output format for pictures
3. Click on "SLICE"
4. Select the video
5. Select the output folder.
6. (Optional) Click on "SLICE" to interrupt
7. Go and check the folder you select for images
